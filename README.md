# CCR & XVA Lab

Small Streamlit project built to understand how OTC derivatives exposure translates into XVA risk metrics.

The app includes exposure simulation, simplified CVA/FVA/MVA calculations, deterministic CVA stress tests and a spread-only CVA VaR module.

Claude Code was used as a development tool during the implementation.

Live app: [CCR & XVA Lab](https://ccr-xva-lab.streamlit.app/)
